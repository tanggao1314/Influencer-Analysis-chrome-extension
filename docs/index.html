🔌 我的 Chrome 插件 - 安装指南

⬇️ 点击下载最新版本

📋 安装步骤（2 分钟搞定）

第 1 步：下载插件
点击上面的“点击下载最新版本”按钮，下载 .zip 文件。
下载后解压，你会看到一个文件夹，里面包含 manifest.json 等文件。
第 2 步：打开 Chrome 扩展管理页面
在 Chrome 浏览器地址栏输入：
chrome://extensions/
按回车键进入。
第 3 步：开启开发者模式
在页面右上角，找到 “开发者模式” 开关，点击打开（变成蓝色）。
第 4 步：加载插件
点击页面左上角的 “加载已解压的扩展程序” 按钮。
在弹出的窗口中，选择你在第 1 步解压出来的 文件夹。
点击“选择文件夹”。
✅ 完成！
插件图标会出现在 Chrome 右上角，点击即可使用。

⚠️ 注意事项

  • 安装时 Chrome 会提示“该扩展程序未在 Chrome 商店中发布”，点击 “保留” 即可。
  • 每次更新插件，需要重新下载最新版本，然后卸载旧版本，重新加载新文件夹。
  • 如果遇到问题，请确认解压后的文件夹里包含 manifest.json 文件。
