⚡ 适用于 Chrome / Edge / 基于 Chromium 的浏览器

📋 安装步骤 · 30秒搞定

📦 第 1 步：下载插件包
点击上方按钮下载 .zip，解压后得到文件夹（内含 manifest.json）。
📁 解压后文件夹结构
⚙️ 第 2 步：打开扩展管理页面
地址栏输入 chrome://extensions/ 并回车。
⚙️ 管理所有扩展
🔄 第 3 步：开启开发者模式
右上角 “开发者模式” 开关打开（蓝色）。
🔄 开启后加载本地扩展
📂 第 4 步：加载已解压的扩展
点击 “加载已解压的扩展程序”，选择解压后的文件夹。
📂 选择包含 manifest.json 的文件夹
✅ 完成！
插件图标出现在浏览器右上角，点击即可使用。
🧩 点击图标打开分析面板

⚠️ 注意事项

  • 安装时 Chrome 提示“未在 Chrome 商店发布”，点击 “保留” 即可。
  • 更新插件：重新下载最新版，卸载旧版，重新加载新文件夹。
  • 确保解压后文件夹内包含 manifest.json，否则无法加载。
  • 部分企业浏览器可能限制安装，请使用个人 Chrome 或 Edge。
